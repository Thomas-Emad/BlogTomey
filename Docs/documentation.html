# Users | Dashboard

This is a simple page that shows you as an administrator of the site iMac all users so that you can change the permissions of anyone or delete their account.

## What are Permissions?

it the permissions that people have.

  1. Owner: The owner can do everything, such as modifying the page, changing the permissions of others, adding an article, deleting any article, deleting any comment.
  2. Admin: admin can do everything like the owner, but he cannot modify the design of the page in terms of information, and he cannot change the powers of anyone or delete him.
  3. Editor: It is the editor that can only add a new article, modify it and delete it.
  4. User: He is the person who visited the site, but would like to create an account so that he can add a comment with his name only.
